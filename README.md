# 📊 Jobs and Salaries in Data Science: Análise Exploratória (EDA)

## 🎯 Objetivo do Projeto
Este projeto realiza uma Análise Exploratória de Dados (EDA) para mapear o cenário global de salários na área de Ciência de Dados. O foco foi entender a estrutura salarial, identificar padrões de mercado e suportar decisões baseadas em dados através do tratamento e inspeção de informações reais.

## 🛠️ Tecnologias e Ferramentas
* **Linguagem:** Python
* **Manipulação e Limpeza:** Pandas, NumPy
* **Ambiente de Desenvolvimento:** Google Colab / Jupyter Notebook

## ⚙️ Metodologia Aplicada (Pipeline)
1. **Extração:** Importação de dataset público (Kaggle).
2. **Inspeção e Tratamento:** Limpeza de dados, verificação de valores nulos, tipagem e consistência estrutural.
3. **Análise de Variáveis:** Identificação e separação de variáveis qualitativas e quantitativas.
4. **Análise Exploratória (EDA):** Mapeamento de padrões relacionados a cargos, localização e variação salarial por nível de experiência.

## 💡 Principais Insights (Resultados Quantitativos)
* 💰 **Variação Salarial:** Foi identificada uma variação de **91%** nos salários entre os níveis Júnior (Entry-level) e Sênior.
* 🌍 **Padrões de Mercado:** Exploração detalhada da distribuição salarial e cargos com maior demanda global.
